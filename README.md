# 📊 Amazon Sales Data Analysis

## 🔍 Project Overview

This project performs **Exploratory Data Analysis (EDA)** on an Amazon sales dataset to uncover sales patterns, high-performing product categories, and key business insights. The goal is to transform raw sales data into actionable insights that can support data-driven decision-making.

---

## 🎯 Objectives

* Clean and preprocess raw sales data
* Perform exploratory data analysis using Python
* Identify sales trends and category performance
* Create visualizations for business insights
* Highlight high-performing products and seasonal patterns

---

## 🛠️ Tech Stack

* **Python**
* **Pandas**
* **NumPy**
* **Matplotlib**
* **Seaborn**
* **Jupyter Notebook**

## 🚀 How to Run the Project

1. Clone the repository

```bash
git clone https://github.com/GarimaKhirwar/amazon-sales-data-analysis.git
```

2. Navigate to the project folder

```bash
cd amazon-sales-data-analysis
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Open the notebook

```bash
jupyter notebook
```
